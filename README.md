A simple Streamlit app that fetches live Bitcoin price data from RapidAPI and uses Google's Gemini model to generate a beginner-friendly technical analysis.

The app:
- Pulls the last 7 days of Bitcoin prices from the Coinranking API (via RapidAPI)
- Sends those prices to a Gemini model using `google-generativeai`
- Returns a human-readable technical analysis (price overview, moving averages, RSI, MACD, buy/sell suggestion, etc.) :contentReference[oaicite:1]{index=1}

---

## Features

- Live Bitcoin price history (last 7 days)
- Technical analysis generated by Gemini (`gemini-pro-latest` model)
- Beginner-friendly explanation of indicators
- Clean Streamlit UI (`Analyze` button + text area output)
- Secrets handled securely using `.streamlit/secrets.toml` (no keys hardcoded in code)

---

## Tech Stack

- **Language:** Python
- **Framework:** Streamlit
- **LLM:** Google Gemini (via `google-generativeai`)
- **Data Source:** Coinranking API (RapidAPI)
- **HTTP:** `requests`
